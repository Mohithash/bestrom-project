# BestROM — Design Philosophy

**Battery is the most important feature.**  
**Look:** Nothing OS–like **ultra minimal monochrome** — **black + grey dots only**.

---

## The problem with modern Android

Too much budget goes to cosmetic skins, animations, artworks, feature checklists, and placebo “optimizations.” That steals **smoothness**, **fluidity**, and **battery**.

## Our answer

**BestROM** = pure **AOSP Android 17** daily driver:

1. **Battery** first  
2. **Smoothness** by staying light  
3. **AOSP-compliant** product layer  

> Best battery + real smoothness — without the cosmetic tax.

## Monochrome law (like Nothing, darker for battery)

| Allowed | Role |
|---------|------|
| **Black `#000000`** | Only background |
| **Grey dots / type** | Only brand marks + UI text greys |

**No brand violet, cyan, or rainbow.** Identity is silence + dots, not color theater.

| Default | Spec |
|---------|------|
| Canvas | Pure black |
| Theme | Dark only |
| Logo / wordmark | **Grey dotted** letters |
| Accents | Greys only (dim → bright) |

We take **Nothing OS** taste: ultra minimal, empty space, **dot letter craft**.  
We are **not** Nothing: base is **AOSP 17**, and we stay **black + grey** (no light product skin).

## Three questions (every default)

1. Help **battery**?  
2. Keep **smooth / fluid**?  
3. Stay **AOSP-maintainable**?

If it’s only trailer chrome (or brand color noise), it does not ship as default.

## Refuse / build

| Refuse | Build |
|--------|-------|
| Heavy skins | Pure AOSP 17 |
| Multicolor brand systems | **Black + grey only** |
| Solid neon logos | **Grey dotted** B / BESTROM |
| Placebo toggles | Honest thrifty controls |

## One line

> *Black. Grey dots. Ultra minimal. More battery. Still Android — pure AOSP 17.*
