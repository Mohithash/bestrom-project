# About BestROM

BestROM is a **hobby** custom ROM based on **pure AOSP Android 17**.

It aims for long battery life and a simple **black + grey** interface — not a heavy skin or a long feature list.

| | |
|--|--|
| Base | AOSP 17 (`android-17.0.0_r1`) |
| Focus | Battery · smoothness · small surface area |
| UI | Dark by default · monochrome brand (black / grey dots) |
| First device | POCO F6 / Redmi Turbo 3 (`peridot`) |
| Downloads | [sourceforge.net/projects/bestrom](https://sourceforge.net/projects/bestrom/) |

## Status

Work in progress. Brand and trees are public; a full A17 daily-driver build for peridot is still being brought up.
