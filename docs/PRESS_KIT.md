docs are public. A17 bring-up is first-wave work; VoltageOS on A16 remains a
parallel reference while BestROM A17 ports land.

## Talking points (interviews / comments)

1. **Battery is the product**, not a settings afterthought.  
2. **Pure AOSP** — maintainable, not a 40-fork skin OS.  
3. **Monochrome** — black canvas + grey dotted identity (AMOLED thrift).  
4. **Peridot first** — honest WIP, not 50 device vaporware.  
5. **Open trees** — GitHub for brand, manifest, device, vendor.  

## What not to claim (yet)

- “Stable daily driver A17” before boot + RIL/camera baseline  
- “Better battery than stock” without user data  
- “Official Nothing OS” / clone (we are **inspired**, not affiliated)
