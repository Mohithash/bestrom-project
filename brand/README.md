# BestROM branding kit

Assets and rules for **Aurora Night** — the visual face of a **battery-first** ROM.

Philosophy lives in the hub: [docs/PHILOSOPHY.md](../docs/PHILOSOPHY.md)

## Promise

| Priority | Meaning |
|----------|---------|
| 1. Battery | Pure black canvas, sparse light, thrifty defaults |
| 2. Smoothness | No cosmetic GPU tax |
| 3. AOSP | Brand overlays — not a heavy skin stack |

## Rules for new art

1. **Primary field = `#000000`** — maximize off pixels on OLED  
2. **Accents are sparks** — violet `#7C3AED` / cyan `#22D3EE` only in small areas  
3. **No full-screen neon**, no busy artwork walls, no light themes as brand defaults  
4. **No fake “performance” chrome** — restraint *is* the brand  
5. Prefer flat / geometric marks over 3D glossy skins
